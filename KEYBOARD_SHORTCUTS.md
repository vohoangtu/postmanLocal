# Keyboard Shortcuts

Danh sách đầy đủ các keyboard shortcuts trong PostmanLocal.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+K` (Windows/Linux)<br>`Cmd+K` (Mac) | Command Palette | Mở Command Palette để tìm kiếm và thực hiện các actions |
| `Ctrl+B` (Windows/Linux)<br>`Cmd+B` (Mac) | Toggle Left Panel | Bật/tắt left panel (collections, history, etc.) |
| `Ctrl+/` | Help | Hiển thị danh sách keyboard shortcuts |

## Request Builder

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+S` | Save Request | Lưu request vào collection |
| `Ctrl+Enter` | Send Request | Gửi request hiện tại |
| `Ctrl+Shift+S` | Save As | Lưu request với tên mới |
| `Ctrl+N` | New Request | Tạo request mới |
| `Ctrl+W` | Close Tab | Đóng tab hiện tại |
| `Ctrl+T` | New Tab | Tạo tab mới |

## Editor Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+F` | Find | Tìm kiếm trong editor |
| `Ctrl+H` | Replace | Tìm và thay thế |
| `Ctrl+/` | Toggle Comment | Comment/uncomment dòng hiện tại |
| `Ctrl+Shift+K` | Delete Line | Xóa dòng hiện tại |
| `Alt+Up/Down` | Move Line | Di chuyển dòng lên/xuống |
| `Ctrl+D` | Duplicate Line | Duplicate dòng hiện tại |

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+1` | Collections | Mở Collections panel |
| `Ctrl+2` | History | Mở History panel |
| `Ctrl+3` | Environments | Mở Environments panel |
| `Ctrl+4` | Schemas | Mở Schemas panel |
| `Ctrl+5` | Mock Server | Mở Mock Server panel |
| `Ctrl+6` | Templates | Mở Templates panel |
| `Ctrl+7` | Workspaces | Mở Workspaces panel |

## Response Viewer

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Shift+C` | Copy Response | Copy toàn bộ response |
| `Ctrl+Shift+B` | Copy Body | Copy chỉ response body |
| `Ctrl+Shift+H` | Copy Headers | Copy chỉ response headers |
| `Ctrl+R` | Format JSON | Format JSON response (nếu là JSON) |

## Tips

- Hầu hết các shortcuts có thể được customize trong Settings (coming soon)
- Shortcuts có thể khác nhau giữa Windows/Linux và Mac
- Sử dụng `Ctrl+K` để tìm kiếm bất kỳ action nào
